# Imminence-x-CDA.AmpSum
Human reaction to approaching threat or uncertainty. Imminence (X-axis) represents how close an event is to occurring. Participant ID (Y-axis) separates individual subjects. CDA AmpSum (Z-axis) represents the cumulative magnitude of the neural response.

Note: 
- X = Imminence (how scary/ known/ unkown the threats are)
- Y = ID
- Z = CDA. AmpSum (overall reaction (peak reaction + #n reaction))

- Purpose of chart: Observation of human reation toward threats (Unknown or not)
